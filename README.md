# JSON2CRM

Приложение для отправки данных в CRM PELETON методом POST в формате JSON.

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

Создайте файл `config.py` с настройками API:

```python
# Конфигурация API
API_URL = "..."
```

> ⚠️ Файл `config.py` добавлен в `.gitignore` — не забудьте настроить его локально.

## Запуск

```bash
python main.py
```

## Сборка в EXE

```bash
json2crm_installer_2.bat
```

## Поля для заполнения

- **Телефон** — номер телефона клиента
- **Комментарий** — комментарий к заявке

## Неизменяемые поля

- **Статус**: `Перевели на оператора` (триггер для заведения заявки)
- **Источник**: `394` (ID проекта)
- **createdAt**: текущее время (автоматически при отправке)
